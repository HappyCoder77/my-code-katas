# 🥋 Professional Django & Docker Learning Lab

Welcome to my personal training ground for **Django** and **Cloud-Native Development**. This repository is a collection of "Katas"—repeated exercises designed to master the art of building scalable, secure, and professional web applications.

## 🚀 The Philosophy
In martial arts, a *Kata* is a structured pattern of movements. In software, it's about **mechanizing best practices** (Dockerization, non-root security, clean settings) so they become second nature.
